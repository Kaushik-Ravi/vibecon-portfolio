Case Study 03

Pune Tree Dashboard

Large-scale geospatial intelligence for urban forestry planning and climate analysis.

Role: Platform architecture and geospatial product design

Core stack: MapLibre, TypeScript, PostGIS, Supabase, satellite overlays

Status: Production deployment

Problem

Urban forestry decisions often rely on fragmented records and static reports, making it difficult to prioritize interventions ward-by-ward.

Approach

This platform integrates tree census records, map interaction, filtering, and environmental layers into an analysis workspace for planners and policy teams.

  • 1.79M tree records with multi-dimensional filters
  • Ward-level analysis and planning support
  • Temperature and canopy context for climate-focused decisions

Key Constraints and Fixes

  • Climate action teams requested explicit street-tree filtering not available in the initial dataset
  • Street-tree layer added using OpenStreetMap road buffers and tree-within-buffer extraction
  • Heavy query latency reduced using local storage cache, Redis, and frontend state optimization

Key Technical Decisions

  • Preserved census-scale source table and layered analytical filters on top
  • Added OSM-derived spatial buffering pipeline to create street-tree focused views
  • Combined client cache, Redis cache, and frontend state strategy to reduce repeated heavy queries

Media Gallery

Outcomes

  • Unified large-scale tree intelligence in one operational interface
  • Improved discoverability of ward-level green infrastructure signals
  • Built foundation for data-backed planting and policy planning
  • Primary tree inventory scale: 1.79M records from Pune tree census datasets

Validation note: overlays include NDVI and tree-cover change context from 2019-2025, with satellite basemaps and census cross-check workflows.

Learnings

  • User-requested filters can redefine product value faster than backend feature expansion
  • Caching is not optional at this scale; it is foundational to usable map analytics
  • Cross-validating satellite overlays with census context improves planner trust
